Zig Community Mirror

This is a community-hosted mirror for the Zig programming language, designed to provide faster, local access to Zig downloads.

For a complete list of official and community mirrors, please visit the official Zig website: ziglang.org/download/community-mirrors/.

Mirror Status

This mirror is operational. Files are served from a local cache. If a requested file is not in the cache, it will be automatically and securely downloaded from the official upstream source and cached for future requests.

About the Software

This mirror is powered by go-mirror-zig, a custom, open-source server implementation written in Go.

Support and Issue Reporting

If you encounter any issues with this mirror, such as slow downloads or failed requests, please report them to the mirror's maintainer. The maintainer's contact information is available at github.com/ziglang/www.ziglang.org/assets/community-mirrors.ziggy

If you encounter any odd behavior related to the caching server software, such as incorrect checksums, wrong status codes, or other unexpected behavior, please report them by opening an issue on the project's GitHub repository: github.com/savalione/go-mirror-zig/issues
